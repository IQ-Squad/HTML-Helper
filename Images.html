How to add images?

Images can be easily inserted at any section in an HTML page. To insert image in an HTML page, use the (img) tags. It is an empty tag, containing only attributes since the closing tag is not required. Just keep in mind that you should use the (img) tag inside (body) â€¦ (/body) tag. The src attribute is used to add the image source i.e. URL of the image. The alt attribute is for adding alternate text, width for adding width, and height for adding the height of the image.
