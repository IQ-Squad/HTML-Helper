How to change fontstyle?

To change the text font in HTML, use the style attribute. The style attribute specifies an inline style for an element. The attribute is used with the HTML (p) tag, with the CSS property font-family, font-size, font-style, etc. HTML5 do not support the (font) tag, so the CSS style is used to change font. The (font) tag deprecated in HTML5. Just keep in mind, the usage of style attribute overrides any style set globally. It will override any style set in the HTML (style) tag or external style sheet.
